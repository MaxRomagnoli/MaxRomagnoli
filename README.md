# [Max Romagnoli website](https://www.maxromagnoli.com)
Story and portfolio of Max Romagnoli

## Developed using:
- Bootstrap 5
- Swiper.js
- JQuery.js

## Features
- Localization (en, it)
- Dark Mode
- Chat helper 24/24 7/7
- Search bar
